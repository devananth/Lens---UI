Buttons

Buttons allow users to take actions, and make choices, with a single tap. These are also called as Call to Action.

Example of Primary Buttons

To generate a primary button, use the base class btn.btn-primary and add any of the following supporting classes to get different variations for a primary button- btn-solid, btn-outline, btn-link, btn-icon, and btn-text-icon.

Example of Secondary Buttons

To generate a primary button, use the base class btn.btn-secondary and add any of the following supporting classes to get different variations for a primary button- btn-solid, btn-outline, btn-link, btn-icon, and btn-text-icon.

Example of Disabled Buttons

You can make a button disabled by adding .btn-disabled

Example of Different Size of Buttons

You can use .btn-sm and .btn-lg to generate small and large buttons.

Example of Floating Action Buttons

Floating Action Buttons are used to perform the most common action on the screen. You can use .btn-float-action.
