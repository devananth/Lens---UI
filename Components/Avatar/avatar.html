Avatars

An avatar is an icon or image which is used to represent the person behind it.
Eg: In a profile pic of any social media application.

Avatars are available in 4 different sizes. You can add the respective classes to get the your avatar ready.

Example of Rounded Image Avatar

Rounded image avatars are mostly used a profile pic in many social media.

Example of Square Image Avatar

Square image avatars can be created by just adding .rounded .

Example of Text Avatar

Text avatars are used to represent the initials of the user's First Name or Last Name. You can use .avatar-txt to create text avatars.
